About the Project

Spud is an accessible gaming controller designed for users with one functioning hand. This capstone project focuses on inclusivity, comfort, and modular interaction design.

Features

  • One-Handed Operation: Comfortable, wearable input system.
  • Custom Input Mapping: Tailored controls for any user.
  • Modular Design: Adjustable components for flexibility.
